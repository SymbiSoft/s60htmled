S60 HTML Editor

Руководство пользователя

S60HTMLEditor представляет собой текстовый редактор, предназначенный для ввода и редактирования HTML-файлов на смартфонах под управлением ОС Symbian (S60 Ed.3).

Редактор облегчает ввод тегов и entities HTML при помощи системы меню.

Установка

Для работы приложения скачайте и установите на смартфон Python for S60 и пакет appuifw2.

Установка SIS-файла

Программа может быть установлена как самостоятельное приложение (только на S60 3rd edition). Скачайте свежую версию и установите ее обычным способом. Программа и пакет appuifw2 должны быть установлены на тот же диск, куда установлен Питон!

Установка скрипта

Скачайте файл s60htmled.py с сайта и скопируйте его в папку \Python на телефоне (на том же диске, где у вас установлен Питон). Запустите интерпретатор, нажмите кнопку Функции, выберите в меню Run script и выберите в появившемся списке s60htmled.py.

Главное окно программы

В главном окне программы наверху отображается имя редактируемого файла, под которым расположен индикатор текущего положения курсора, который показывает текущее местоположение курсора в процентах относительно общего размера файла.

В нижней части окна отображаются текущие назначения двух функциональных клавиш телефона. Левая клавиша вызывает главное меню, правая вызывает список тегов.

Остальная часть экрана служит для отображения редактируемого текста.

Перемещение по тексту, горячие клавиши

Для перемещения по тексту используются клавиши со стрелками, которые перемещают курсор на символ влево/вправо или на строку вверх/вниз. С помощью клавиши Shift (на некоторых моделях решетка) со стрелками текст может быть выделен и скопирован в буфет обмена, вставлен или удален обычным для телефона способом.

Клавиаша Call (зеленая трубка) используется в редакторе в качестве функциональной: она на некоторое время изменяет назначение других клавиш:

  • Call + стрелка влево - в начало строки
  • Call + стрелка вправо - в конец строки
  • Call + стрелка вверх - страница вверх
  • Call + стрелка вниз - страница вниз
  • Call + клавиша выбора - открывает меню быстрого перехода, включающего в себя переход в начало текста, конец текста или на заданную строку.
  • Call + правая функциональная клавиша - открывает меню вставки частоупотребимых HTML-символов.

При нажатии на зеленую трубку вместо индикатора позиции курсора появляется надпись Func, исчезающая через полторы секунды или при нажатии любой клавиши.

Главное меню

Главное меню вызывается левой функциональной клавишей и состоит из следующих пунктов:

  • File - Операции с файлами
    • Open - Открыть файл
    • Save - Сохранить файл
    • Save as - Сохранить файл под другим именем
    • New - Новый файл
    • New from template - Новый файл из шаблона (доступно два варианта: простой HTML и HTML 4.01 Transitional)
    • View in browser - Позволяет открыть редактируемый файл для просмотра во встроенном браузере телефона (фактически файл открывается приложением, которое ассоциировано с данным типом файла в системе).
  • Edit - Функции редактирования
    • Undo - Отмена последней команды редактирования
    • Cut - Вырезать в буфер обмена
    • Copy - Скопировать в буфер обмена
    • Paste - Вставить из буфера обмена
    • Select all - Выделить весь текст
    • Select none - Снять выделение
  • Search - Функции поиска и замены
    • Find forward - Поиск от текущей позиции курсора к концу текста
    • Find backward - поиск от текущей позиции курсора к началу текста
    • Replace - Найти и заменить (поиск производится от текущей позиции курсора к концу текста, заменяется первое вхождение)
  • Help - Справка
    • Help - Вывод справки
    • About - Информацию о версии программы и авторских правах
  • Exit - Выход из программы

Вставка тегов

Для вставки тега HTML нажмите правую функциональную клавишу и выберите в открывшемся списке нужный тег. В списке работает поиск - достаточно набрать первый символ, чтобы остались только теги, начинающиеся с набранного символа.

Можно выбрать пункт Custom, и ввести любой тег. Таким образом можно также вводить теги, отсутствующие в списке.

Редактор вставляет сразу открывающий и, если нужно, закрывающий теги. Также можно выделить текст и выбрать тег - выделенный текст будет обрамлен выбранным тегом.

Если того требует синтаксис, после выбора тега редактор запросит имена атрибутов (например, для тегов <A> или <IMG>). Вводите требуемые атрибуты, затем нажмите Отменить. Атрибуты вставляются с пустыми значениями.
